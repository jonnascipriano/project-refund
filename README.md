# 💸 Refund - Sistema de Solicitação de Reembolso

Um aplicativo web moderno desenvolvido para simplificar o registro e a gestão de despesas para solicitações de reembolso. Com uma interface limpa e focada na experiência do usuário (UX), a aplicação permite categorizar gastos e calcula automaticamente o valor total a ser reembolsado.

## 🔖 Sobre o Projeto

O **Refund** nasceu da necessidade de praticar a manipulação avançada do DOM com JavaScript, lidando com formulários, eventos dinâmicos, cálculos matemáticos em tempo real e formatação de strings (moeda). O design foi pensado especialmente para dispositivos móveis, trazendo uma paleta de cores escura (Dark Mode) elegante e cartões de informação claros e legíveis.

## 🚀 Tecnologias Utilizadas

Este projeto foi construído utilizando as seguintes tecnologias:

* ![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=for-the-badge&logo=html5&logoColor=white) - Estrutura e semântica web.
* ![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=for-the-badge&logo=css3&logoColor=white) - Estilização, layout responsivo e variáveis de cores.
* ![JavaScript](https://img.shields.io/badge/JavaScript-323330?style=for-the-badge&logo=javascript&logoColor=F7DF1E) - Lógica de negócios, cálculos de despesas e manipulação do DOM.

## ⚙️ Funcionalidades

- [x] Cadastro de novas despesas informando: Descrição, Categoria (Alimentação, Transporte, Hospedagem, etc.) e Valor.
- [x] Geração dinâmica de itens na lista de solicitações.
- [x] Cálculo automático e em tempo real do valor total das despesas.
- [x] Formatação automática de valores para a moeda local (Real Brasileiro - R$).
- [x] Função de exclusão de despesas individuais da lista (com recálculo automático do total).
- [x] Interface totalmente responsiva e adaptada para uso no celular.
